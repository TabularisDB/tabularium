# Tabularium

> Self-hosted Plugin-Registry für das **TabularisDB**-Ökosystem. Gebaut mit [Bun](https://bun.sh), [Elysia](https://elysiajs.com), [SvelteKit](https://kit.svelte.dev) und [Drizzle ORM](https://orm.drizzle.team).

🌐 **In deiner Sprache:** [English](README.md) · [Deutsch](README.de.md) · [Español](README.es.md) · [Français](README.fr.md) · [Italiano](README.it.md) · [中文](README.zh-CN.md)

📚 **Docs:** [Tabularium-Doku](docs/) (Docsify, deploybar auf Codeberg Pages / git-pages.org)

## Was ist Tabularium?

Ein Verzeichnis für Plugins (oder beliebige Release-Artefakte) mit Web-UI, OAuth-Submission-Flow, OpenAPI-Schnittstelle und integriertem CMS. Autor:innen verbinden ihren GitHub-/GitLab-/Gitea-Account, wählen ein Repo aus, und Tabularium installiert den Release-Webhook — jeder neue Tag wird automatisch zum Release im Verzeichnis.

## Features

- 🧩 **Multi-Provider-Submission** — GitHub, GitLab, Gitea (beliebige Instanzen)
- 🪄 **Install-Wizard** — Kaltstart ohne Datenbank, Bootstrap-Admin-Passwort im Log
- 📝 **Integriertes CMS** — Markdown-Seiten mit Widgets, Übersetzungen pro Sprache
- 🎨 **Branding** — Name, Farben, Logo, Favicon, Analytics, Indexierungs-Policy
- 🌍 **6 UI-Sprachen** — English, Deutsch, Español, Français, Italiano, 中文 — vom Admin konfigurierbar
- 🗄 **Multi-Dialekt** — SQLite, Postgres oder MySQL (auto-erkannt aus `DATABASE_URL`)
- 🚦 **Feature-Toggles** — Submissions oder Requests ohne Redeploy abschalten
- 🪵 **Audit-Log** — jede Admin-Aktion mit Akteur + IP + Ziel

## Schnellstart

```bash
git clone https://codeberg.org/NewtTheWolf/Tabularium
cd Tabularium
bun install
docker compose -f compose.dev.yml up -d   # postgres + dragonfly (optional)

# in einem Terminal
cd apps/backend && bun --hot src/index.ts

# in einem zweiten
cd apps/frontend && bun dev
```

Öffne `http://localhost:5180` — der Wizard führt durch die Datenbankeinrichtung, lässt Migrations laufen, seedet Default-CMS-Seiten und befördert den Bootstrap-Account zum echten Admin.

Das Bootstrap-Passwort steht im Startlog:

```
==========================================
 Tabularium install wizard
 → http://localhost:3000/welcome
 Bootstrap login:
   admin@example.com
   <auto-generated-password>
==========================================
```

## Stack

| Schicht | Tech |
|---------|------|
| Runtime | [Bun](https://bun.sh) (≥ 1.3) |
| HTTP | [Elysia](https://elysiajs.com) + TypeBox |
| DB | [Drizzle ORM](https://orm.drizzle.team) — sqlite / postgres-js / mysql2 |
| Cache | `Bun.redis` (Dragonfly-kompatibel) oder in-memory |
| Frontend | SvelteKit (SPA) + [Eden Treaty](https://elysiajs.com/eden/treaty/overview) end-to-end Typsicherheit |
| i18n | [Paraglide JS](https://inlang.com/m/gerre34r/library-inlang-paraglideJs) |
| Docs | [Docsify](https://docsify.js.org) — kein Build-Step |

## Projektstruktur
