# `@tabularium/api`

Elysia HTTP server + install wizard. Drizzle ORM against SQLite / Postgres / MySQL (auto-detected from `DATABASE_URL`).

## Develop

From the repo root:

```bash
bun install
just dev   # starts minikube + tilt up (full dev stack)
```

The api is port-forwarded to `localhost:3000`. The first boot prints a one-time bootstrap login:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  FIRST-RUN SETUP
    Email:    admin@example.com
    Password: <auto-generated>
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

Walk through `/init` to provision the DB, then sign in.

## Test

```bash
bun test
```

Tests run against an in-memory SQLite by default. The helpers in `tests/helpers.ts` reset state between cases.

## Layout

```
src/
  index.ts              Elysia bootstrap (loads file-router under src/routes)
  routes/               file-routed HTTP surface
    api/...             public + admin JSON API
    auth/...            OAuth callbacks + session
    webhooks/...        per-provider release webhooks
    well-known/...      JWKS at /.well-known/registry-key.json
  middleware/           auth, rate-limit, audit, logger
  lib/                  domain logic (manifest, release-integrity, oauth-tokens, …)
  db/                   schema + migrations + seeds (multi-dialect)
```

## OpenAPI

Live spec at `GET /openapi/json`. Swagger UI at `/openapi`. The typed client in `packages/client` is generated from this surface via Eden Treaty.
