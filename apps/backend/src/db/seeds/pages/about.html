Self-hosted plugin registry

About Tabularium

Tabularium is a self-hosted plugin registry for the TabularisDB ecosystem. Bring your own git provider, let authors submit through OAuth, and keep release metadata fresh through webhook-driven ingestion.

Bring your own git host

Works against any GitHub, GitLab, or Gitea instance the operator wires up. Plugins live in their own repos; the registry only indexes the metadata it needs.

Release-driven

Configure one webhook per provider and every new tag refreshes the version table automatically. No manual republishing, no artifact storage to babysit.

Typed API

An Eden Treaty client, an OpenAPI spec, and predictable JSON. Build whatever surface you want on top — CLI, dashboard, or another frontend entirely.

For plugin authors

Sign in with your git provider, pick a repo, and submit. A small .tabularium manifest at the repo root describes the plugin — name, category, tags, icon, screenshots. From there releases flow in on their own.
