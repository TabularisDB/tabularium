# Tabularium

> Self-hosted plugin registry for the TabularisDB ecosystem.

- Submit plugins from GitHub, GitLab or Gitea
- Built-in CMS, branding, audit log, feature flags
- 6 UI languages out of the box
- One-binary deploy — SQLite, Postgres or MySQL
